# NoteApp-MVVM

> A simple and minimalistic Notes taking app using LiveData, ViewModel and Room databases in Java

---

****Concepts Covered***
- MVVM Architecture
- View Model
- Live Data & Flow (Recently Added to get single Note)
- Room Database (using Dao)
